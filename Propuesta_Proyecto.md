# Propuesta de Proyecto: VisorMD V2 Full Stack

## Introducción
Este proyecto propone la evolución de una herramienta de visualización de documentación estática a una aplicación Full Stack robusta.

## Descripción del problema
La gestión de documentación técnica en equipos de desarrollo a menudo carece de interactividad y persistencia centralizada.

## Objetivos
- Implementar una base de datos PostgreSQL para almacenamiento.
- Desarrollar una API REST documentada con Swagger.
- Crear una interfaz moderna en React.
